# fiken-go — Plan A: Bootstrap & Codegen Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Take the empty `fiken-go` repo to a state where `nix develop` works, pre-commit hooks are installed and green, the Fiken OAS spec is vendored, and all five codegen / scaffold tools produce their outputs cleanly. After Plan A, the repo compiles and CI gates pass with no business logic yet.

**Architecture:** Nix flake provides every dev tool. `prek` enforces hooks. `ogen` generates the typed Fiken client from a vendored YAML. Four small Go tools at `cmd/fiken-*` produce build-time fixtures from the same YAML: `fiken-spec-update` fetches and diffs upstream, `fiken-mutating-gen` emits `ops/mutating.gen.go`, `fiken-i18n-scaffold` seeds an editor template, and `fiken-spec-lint` checks unit/format declarations. Everything is bit-reproducible: each commit is gated by hooks identical to CI.

**Tech Stack:** Go 1.26, Nix flake (nixpkgs-unstable), `github.com/ogen-go/ogen`, `github.com/j178/prek`, stdlib (`net/http`, `crypto/sha256`, `gopkg.in/yaml.v3`), `golangci-lint`, `difftastic`.

**Spec reference:** `docs/superpowers/specs/2026-05-15-fiken-go-design.md` (this plan implements §"Execution policy" hard rules + Implementation steps 1–3).

**Module path:** `github.com/kradalby/fiken-go`.

````
### Task 2: `flake.nix` and `.envrc` direnv wiring

**Files:**
- Create: `flake.nix`

- [ ] **Step 2.1: Write `flake.nix`**

```nix
{
  description = "fiken-go — Go library, CLI, and MCP server for the Fiken API";

  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixpkgs-unstable";
    flake-utils.url = "github:numtide/flake-utils";
  };

  outputs =
    {
      self,
      nixpkgs,
      flake-utils,
    }:
    # Per-system outputs (packages, devShells, formatter) are merged
    # via // with system-agnostic outputs (overlays) below.
    (flake-utils.lib.eachDefaultSystem (
      system:
      let
        pkgs = import nixpkgs { inherit system; };

        # Prefer Go 1.26 if available, else 1.25. go.mod sets the
        # language version separately; this just selects the
        # toolchain in the devShell and package builder.
        go = pkgs.go_1_26 or pkgs.go_1_25;

        # Build the fiken-go binary. The vendorHash placeholder must be
        # updated after the first `go mod tidy` produces a vendor tree.
        fiken = pkgs.buildGoModule {
          pname = "fiken-go";
          version = "0.0.0-dev";
          src = ./.;
          inherit go;
          # First-time setup: nix build will fail with a hash mismatch and
          # print the correct value — replace this placeholder with that
          # output. Until cmd/fiken/main.go exists in Plan B this also
          # has no entrypoint, so expect `nix build` to error on a
          # different message until then.
          vendorHash = pkgs.lib.fakeHash;
          subPackages = [ "./..." ];
        };
      in
      {
        packages = {
          fiken = fiken;
          default = fiken;
        };

        devShells.default = pkgs.mkShell {
          packages = [
            go
            pkgs.gopls
            pkgs.gofumpt
            pkgs.gotools          # provides goimports
            pkgs.golangci-lint
            pkgs.gotestsum
            pkgs.gotests
            pkgs.difftastic
            pkgs.prek
            pkgs.ogen
            pkgs.prettier
            pkgs.nixfmt-rfc-style
            pkgs.git
          ];

          shellHook = ''
            echo "fiken-go devShell — Go $(${go}/bin/go version | cut -d' ' -f3)"
          '';
        };

        formatter = pkgs.nixfmt-rfc-style;
      }
    ))
    // {
      # System-agnostic outputs live outside eachDefaultSystem so
      # `inputs.fiken-go.overlays.default` resolves correctly for any
      # consumer regardless of their `system`.
      overlays.default = final: prev: {
        fiken = self.packages.${prev.system}.fiken;
      };
    };
}
````

```yaml
# All local hooks run inside `nix develop` via `pre-commit-utils` style entries
# so the tool versions match the flake exactly. We rely on prek (Rust drop-in
# for pre-commit) reading the same file.

repos:
  - repo: https://github.com/pre-commit/pre-commit-hooks
    rev: v6.0.0
    hooks:
      - id: check-added-large-files
        args: ["--maxkb=1024"]
      - id: check-case-conflict
      - id: check-merge-conflict
      - id: check-symlinks
      - id: check-json
      - id: check-toml
      - id: check-xml
      - id: check-yaml
        exclude: '^api/fiken-openapi\.yaml$'
      - id: detect-private-key
      - id: fix-byte-order-marker
      - id: end-of-file-fixer
        exclude: '^api/fiken-openapi\.yaml$'
      - id: trailing-whitespace
        exclude: '^api/fiken-openapi\.yaml$'
      - id: mixed-line-ending

  - repo: local
    hooks:
      - id: nixfmt
        name: nixfmt
        language: system
        entry: nix develop -c nixfmt
        files: '\.nix$'

      - id: prettier
        name: prettier
        language: system
        entry: nix develop -c prettier --write
        files: '\.(json|yaml|yml|md)$'
        exclude: '^(api/|fiken/|.*\.gen\.go$|i18n/locales/.*\.template\.toml$)'

      - id: gofumpt
        name: gofumpt
        language: system
        entry: nix develop -c gofumpt -w
        files: '\.go$'
        exclude: '^fiken/|\.gen\.go$'

      - id: goimports
        name: goimports
        language: system
        entry: nix develop -c goimports -w -local github.com/kradalby/fiken-go
        files: '\.go$'
        exclude: '^fiken/|\.gen\.go$'

      - id: golangci-lint
        name: golangci-lint
        language: system
        entry: nix develop -c golangci-lint run --fix --timeout=5m
        pass_filenames: false
        types: [go]

      - id: codegen-clean
        name: codegen-clean
        language: system
        # Snapshot status before and after `go generate ./...`. Hook
        # fails if generation introduced ANY change (modified tracked
        # file, new untracked file, or removed file). This is more
        # correct than `git diff --exit-code` (misses untracked) and
        # than `test -z "$(git status --porcelain)"` (false-positives
        # on pre-existing untracked files, e.g. the very first
        # `prek run --all-files` before the bootstrap commit lands).
        entry: bash -c 'BEFORE=$(git status --porcelain); nix develop -c go generate ./...; test "$BEFORE" = "$(git status --porcelain)"'
        pass_filenames: false
        stages: [pre-commit]

      - id: go-test
        name: go-test-short
        language: system
        entry: nix develop -c go test -race -count=1 -short ./...
        pass_filenames: false
        stages: [pre-commit]
        types: [go]

      - id: go-test-full
        name: go-test-full
        language: system
        entry: nix develop -c go test -race -count=1 ./...
        pass_filenames: false
        stages: [pre-push]
        types: [go]

      - id: golangci-lint-strict
        name: golangci-lint (no --fix)
        language: system
        entry: nix develop -c golangci-lint run --timeout=5m
        pass_filenames: false
        stages: [pre-push]
        types: [go]
```

Notes for the implementer:

- The `i18n-keys`, `oas-units` hooks land in Plan B after their inputs exist.
- The `vendor-hash` hook lands in Plan D when there's a real vendor / hash file to regenerate.
- `codegen-clean` is enabled here so future commits that touch the spec or scaffold tools always regenerate before commit. On the initial-bootstrap commit there are no `//go:generate` directives yet, so the hook is a no-op.

```go
// Command fiken-spec-update fetches the canonical Fiken OpenAPI spec
// from api.fiken.no, diffs it against the vendored copy, and (on
// confirm) replaces the vendored file plus the provenance record in
// api/SOURCE.txt. Optional --apply skips the prompt.
package main

import (
	"crypto/sha256"
	"encoding/hex"
	"flag"
	"fmt"
	"io"
	"net/http"
	"os"
	"os/exec"
	"time"
)

const (
	specURL  = "https://api.fiken.no/api/v2/docs/swagger.yaml"
	specPath = "api/fiken-openapi.yaml"
	sourceP  = "api/SOURCE.txt"
)

func main() {
	apply := flag.Bool("apply", false, "skip the diff prompt and overwrite the vendored spec")
	diffOnly := flag.Bool("diff-only", false, "print the diff (or '(no changes)') and exit; do not write")
	flag.Parse()

	if err := run(*apply, *diffOnly); err != nil {
		fmt.Fprintf(os.Stderr, "fiken-spec-update: %v\n", err)
		os.Exit(1)
	}
}

func run(apply, diffOnly bool) error {
	tmp, err := os.CreateTemp("", "fiken-spec-*.yaml")
	if err != nil {
		return fmt.Errorf("temp file: %w", err)
	}
	tmpPath := tmp.Name()
	defer os.Remove(tmpPath)
	tmp.Close()

	if err := fetch(specURL, tmpPath); err != nil {
		return err
	}

	same, err := equalFiles(tmpPath, specPath)
	if err != nil {
		return err
	}
	if same {
		fmt.Println("(no changes)")
		return nil
	}

	if err := showDiff(specPath, tmpPath); err != nil {
		return err
	}

	if diffOnly {
		return nil
	}

	if !apply {
		fmt.Print("\nApply changes? [y/N]: ")
		var ans string
		fmt.Scanln(&ans)
		if ans != "y" && ans != "Y" {
			fmt.Println("aborted.")
			return nil
		}
	}

	if err := os.Rename(tmpPath, specPath); err != nil {
		// Cross-device — fall back to copy.
		if err := copyFile(tmpPath, specPath); err != nil {
			return err
		}
	}

	digest, err := computeSHA256(specPath)
	if err != nil {
		return err
	}
	f, err := os.Create(sourceP)
	if err != nil {
		return err
	}
	defer f.Close()
	if err := renderSource(f, specURL, time.Now().UTC().Format(time.RFC3339), digest); err != nil {
		return err
	}
	fmt.Printf("updated %s and %s (sha256 %s)\n", specPath, sourceP, digest)
	return nil
}

func fetch(url, dst string) error {
	resp, err := http.Get(url)
	if err != nil {
		return fmt.Errorf("get %s: %w", url, err)
	}
	defer resp.Body.Close()
	if resp.StatusCode != http.StatusOK {
		return fmt.Errorf("get %s: %s", url, resp.Status)
	}
	f, err := os.Create(dst)
	if err != nil {
		return err
	}
	defer f.Close()
	_, err = io.Copy(f, resp.Body)
	return err
}

func equalFiles(a, b string) (bool, error) {
	if _, err := os.Stat(b); os.IsNotExist(err) {
		return false, nil
	}
	da, err := computeSHA256(a)
	if err != nil {
		return false, err
	}
	db, err := computeSHA256(b)
	if err != nil {
		return false, err
	}
	return da == db, nil
}

func computeSHA256(path string) (string, error) {
	f, err := os.Open(path)
	if err != nil {
		return "", err
	}
	defer f.Close()
	h := sha256.New()
	if _, err := io.Copy(h, f); err != nil {
		return "", err
	}
	return hex.EncodeToString(h.Sum(nil)), nil
}

func renderSource(w io.Writer, url, fetchedAt, sha string) error {
	_, err := fmt.Fprintf(w,
		"source-url: %s\nfetched-at: %s\nsha256:     %s\n",
		url, fetchedAt, sha,
	)
	return err
}

func showDiff(oldPath, newPath string) error {
	bin, err := exec.LookPath("difft")
	args := []string{oldPath, newPath}
	if err == nil {
		cmd := exec.Command(bin, args...)
		cmd.Stdout = os.Stdout
		cmd.Stderr = os.Stderr
		_ = cmd.Run() // difft exits non-zero on diff; ignore.
		return nil
	}
	cmd := exec.Command("git", "--no-pager", "diff", "--no-index", "--color=always", oldPath, newPath)
	cmd.Stdout = os.Stdout
	cmd.Stderr = os.Stderr
	_ = cmd.Run()
	return nil
}

func copyFile(src, dst string) error {
	in, err := os.Open(src)
	if err != nil {
		return err
	}
	defer in.Close()
	out, err := os.Create(dst)
	if err != nil {
		return err
	}
	defer out.Close()
	_, err = io.Copy(out, in)
	return err
}
```

```go
// Command fiken-mutating-gen reads api/fiken-openapi.yaml and writes
// ops/mutating.gen.go: a `package ops` file containing a single
// map[string]bool keyed by operationId in snake_case form.
//
// The mapping rule is purely HTTP-method based: GET/HEAD/OPTIONS map
// to `false` (non-mutating); anything else maps to `true`.
package main

import (
	"flag"
	"fmt"
	"os"
	"sort"
	"strings"

	"gopkg.in/yaml.v3"
)

const (
	defaultSpec   = "api/fiken-openapi.yaml"
	defaultOutput = "ops/mutating.gen.go"
)

func main() {
	specPath := flag.String("spec", defaultSpec, "path to the OAS YAML")
	outPath := flag.String("out", defaultOutput, "output file")
	flag.Parse()

	if err := run(*specPath, *outPath); err != nil {
		fmt.Fprintf(os.Stderr, "fiken-mutating-gen: %v\n", err)
		os.Exit(1)
	}
}

func run(specPath, outPath string) error {
	data, err := os.ReadFile(specPath)
	if err != nil {
		return fmt.Errorf("read spec: %w", err)
	}
	entries, err := parseEntries(data)
	if err != nil {
		return err
	}
	body := render(entries)

	if err := os.MkdirAll(parentDir(outPath), 0o755); err != nil {
		return err
	}
	return os.WriteFile(outPath, []byte(body), 0o644)
}

// classify returns true if the HTTP method mutates state.
func classify(method string) bool {
	switch strings.ToUpper(method) {
	case "GET", "HEAD", "OPTIONS":
		return false
	default:
		return true
	}
}

// parseEntries walks paths.*.{get,post,…}.operationId and emits one
// map entry per operation, key = snake_case(operationId), value =
// classify(method).
func parseEntries(specYAML []byte) (map[string]bool, error) {
	var root struct {
		Paths map[string]map[string]struct {
			OperationID string `yaml:"operationId"`
		} `yaml:"paths"`
	}
	if err := yaml.Unmarshal(specYAML, &root); err != nil {
		return nil, fmt.Errorf("unmarshal spec: %w", err)
	}
	out := make(map[string]bool, 128)
	for _, ops := range root.Paths {
		for method, op := range ops {
			if op.OperationID == "" {
				// Skip path-level parameters etc.
				continue
			}
			out[snake(op.OperationID)] = classify(method)
		}
	}
	if len(out) == 0 {
		return nil, fmt.Errorf("no operationIds found — wrong spec file?")
	}
	return out, nil
}

// snake converts e.g. "getCompanies" → "get_companies" and
// "createSalesV2" → "create_sales_v2".
func snake(s string) string {
	var b strings.Builder
	for i, r := range s {
		if i > 0 && r >= 'A' && r <= 'Z' {
			b.WriteByte('_')
		}
		if r >= 'A' && r <= 'Z' {
			r += 'a' - 'A'
		}
		b.WriteRune(r)
	}
	return b.String()
}

// render emits the generated Go file. Entries are sorted by key for
// stability so `codegen-clean` doesn't fire spuriously.
func render(entries map[string]bool) string {
	keys := make([]string, 0, len(entries))
	for k := range entries {
		keys = append(keys, k)
	}
	sort.Strings(keys)

	// Figure out the longest key so values align in a column.
	maxLen := 0
	for _, k := range keys {
		if len(k) > maxLen {
			maxLen = len(k)
		}
	}

	var b strings.Builder
	b.WriteString("// Code generated by fiken-mutating-gen; DO NOT EDIT.\n\n")
	b.WriteString("package ops\n\n")
	b.WriteString("// mutating maps operation id (snake_case) to whether the underlying\n")
	b.WriteString("// HTTP method mutates server state. Source: api/fiken-openapi.yaml.\n")
	b.WriteString("var mutating = map[string]bool{\n")
	for _, k := range keys {
		pad := strings.Repeat(" ", maxLen-len(k))
		fmt.Fprintf(&b, "\t%q:%s %v,\n", k, pad, entries[k])
	}
	b.WriteString("}\n\n")
	b.WriteString("// IsMutating reports whether op mutates state. Unknown ops are\n")
	b.WriteString("// treated as mutating (fail-closed for read-only mode).\n")
	b.WriteString("func IsMutating(op string) bool {\n")
	b.WriteString("\tif v, ok := mutating[op]; ok {\n")
	b.WriteString("\t\treturn v\n")
	b.WriteString("\t}\n")
	b.WriteString("\treturn true\n")
	b.WriteString("}\n")
	return b.String()
}

func parentDir(p string) string {
	for i := len(p) - 1; i >= 0; i-- {
		if p[i] == '/' {
			return p[:i]
		}
	}
	return "."
}
```

```go
// Command fiken-i18n-scaffold reads api/fiken-openapi.yaml and emits
// i18n/locales/en.template.toml: an editor scaffold with one stanza
// per operation, pre-filled with a `summary` derived from the spec's
// `description` field (or `summary` if present), and empty
// `when_to_use`, `returns`, `example` keys for the author to fill.
//
// The generated file is NOT loaded by the runtime i18n bundle.
// Authors copy or rewrite entries into `i18n/locales/en.toml`.
package main

import (
	"flag"
	"fmt"
	"os"
	"sort"
	"strings"

	"gopkg.in/yaml.v3"
)

const (
	defaultSpec   = "api/fiken-openapi.yaml"
	defaultOutput = "i18n/locales/en.template.toml"
)

func main() {
	specPath := flag.String("spec", defaultSpec, "path to OAS YAML")
	outPath := flag.String("out", defaultOutput, "output file")
	flag.Parse()

	if err := run(*specPath, *outPath); err != nil {
		fmt.Fprintf(os.Stderr, "fiken-i18n-scaffold: %v\n", err)
		os.Exit(1)
	}
}

func run(specPath, outPath string) error {
	data, err := os.ReadFile(specPath)
	if err != nil {
		return err
	}
	entries, err := parseEntries(data)
	if err != nil {
		return err
	}

	if err := os.MkdirAll(parentDir(outPath), 0o755); err != nil {
		return err
	}
	f, err := os.Create(outPath)
	if err != nil {
		return err
	}
	defer f.Close()

	fmt.Fprintln(f, "# Code generated by fiken-i18n-scaffold; not loaded at runtime.")
	fmt.Fprintln(f, "# Copy or rewrite entries from here into en.toml. Edit freely.")
	fmt.Fprintln(f, "")

	keys := make([]string, 0, len(entries))
	for k := range entries {
		keys = append(keys, k)
	}
	sort.Strings(keys)
	for _, k := range keys {
		fmt.Fprint(f, renderEntry(k, entries[k]))
	}
	return nil
}

func parseEntries(specYAML []byte) (map[string]string, error) {
	var root struct {
		Paths map[string]map[string]struct {
			OperationID string `yaml:"operationId"`
			Summary     string `yaml:"summary"`
			Description string `yaml:"description"`
		} `yaml:"paths"`
	}
	if err := yaml.Unmarshal(specYAML, &root); err != nil {
		return nil, err
	}
	out := make(map[string]string, 128)
	for _, ops := range root.Paths {
		for method, op := range ops {
			_ = method
			if op.OperationID == "" {
				continue
			}
			src := op.Summary
			if src == "" {
				src = op.Description
			}
			out[snake(op.OperationID)] = truncateDesc(src)
		}
	}
	return out, nil
}

func renderEntry(op, desc string) string {
	// TOML key column width to keep alignment readable. 4 keys, widest is
	// "when_to_use" (11 chars).
	pad := func(s string) string {
		const want = 11
		return s + strings.Repeat(" ", want-len(s))
	}
	return fmt.Sprintf(
		"[ops.%s]\n"+
			"%s = %s\n"+
			"%s = \"\"\n"+
			"%s = \"\"\n"+
			"%s = \"\"\n"+
			"\n",
		op,
		pad("summary"), tomlString(desc),
		pad("when_to_use"),
		pad("returns"),
		pad("example"),
	)
}

func tomlString(s string) string {
	// Basic-string escaping for ", \, control chars. Good enough for the
	// scaffold; the author can re-quote with multiline `"""` if needed.
	var b strings.Builder
	b.WriteByte('"')
	for _, r := range s {
		switch r {
		case '\\':
			b.WriteString(`\\`)
		case '"':
			b.WriteString(`\"`)
		case '\n':
			b.WriteString(`\n`)
		case '\r':
			b.WriteString(`\r`)
		case '\t':
			b.WriteString(`\t`)
		default:
			b.WriteRune(r)
		}
	}
	b.WriteByte('"')
	return b.String()
}

func truncateDesc(s string) string {
	s = strings.TrimSpace(s)
	if s == "" {
		return ""
	}
	// Prefer first sentence within 200 chars.
	if idx := strings.Index(s, "."); idx >= 0 && idx < 200 {
		return s[:idx+1]
	}
	if len(s) > 200 {
		return s[:200] + "…"
	}
	return s
}

func snake(s string) string {
	var b strings.Builder
	for i, r := range s {
		if i > 0 && r >= 'A' && r <= 'Z' {
			b.WriteByte('_')
		}
		if r >= 'A' && r <= 'Z' {
			r += 'a' - 'A'
		}
		b.WriteRune(r)
	}
	return b.String()
}

func parentDir(p string) string {
	for i := len(p) - 1; i >= 0; i-- {
		if p[i] == '/' {
			return p[:i]
		}
	}
	return "."
}
```

```go
// Command fiken-spec-lint walks api/fiken-openapi.yaml and fails if
// any schema property whose name matches *Date or *At|*Time|*DateTime
// lacks the corresponding OAS `format:` declaration. Catches naive
// datetime strings that would otherwise silently parse as UTC in Go.
package main

import (
	"flag"
	"fmt"
	"os"
	"sort"
	"strings"

	"gopkg.in/yaml.v3"
)

const defaultSpec = "api/fiken-openapi.yaml"

type property struct {
	Type   string `yaml:"type"`
	Format string `yaml:"format"`
}

func main() {
	specPath := flag.String("spec", defaultSpec, "path to OAS YAML")
	flag.Parse()

	violations, err := run(*specPath)
	if err != nil {
		fmt.Fprintf(os.Stderr, "fiken-spec-lint: %v\n", err)
		os.Exit(2)
	}
	if len(violations) > 0 {
		sort.Strings(violations)
		for _, v := range violations {
			fmt.Fprintln(os.Stderr, v)
		}
		fmt.Fprintf(os.Stderr, "\n%d unit-format violation(s)\n", len(violations))
		os.Exit(1)
	}
	fmt.Println("ok")
}

func run(specPath string) ([]string, error) {
	data, err := os.ReadFile(specPath)
	if err != nil {
		return nil, err
	}
	var root struct {
		Components struct {
			Schemas map[string]struct {
				Properties map[string]property `yaml:"properties"`
			} `yaml:"schemas"`
		} `yaml:"components"`
	}
	if err := yaml.Unmarshal(data, &root); err != nil {
		return nil, err
	}
	var out []string
	for schemaName, schema := range root.Components.Schemas {
		for propName, prop := range schema.Properties {
			out = append(out, checkProperty(schemaName, propName, prop)...)
		}
	}
	return out, nil
}

func checkProperty(schema, name string, p property) []string {
	low := strings.ToLower(name)
	switch {
	case strings.HasSuffix(low, "datetime"):
		if p.Format != "date-time" {
			return []string{fmt.Sprintf("%s.%s: name ends in DateTime but format=%q (want date-time)", schema, name, p.Format)}
		}
	case strings.HasSuffix(low, "at"), strings.HasSuffix(low, "time"):
		// Exclude *Date which we handle below.
		if strings.HasSuffix(low, "date") {
			break
		}
		if p.Type != "string" {
			return nil // ints named "createdAt" are unlikely but not our concern
		}
		if p.Format != "date-time" {
			return []string{fmt.Sprintf("%s.%s: name suggests datetime but format=%q (want date-time)", schema, name, p.Format)}
		}
	case strings.HasSuffix(low, "date"):
		if p.Type != "string" {
			return nil
		}
		if p.Format != "date" && p.Format != "date-time" {
			return []string{fmt.Sprintf("%s.%s: name ends in Date but format=%q (want date or date-time)", schema, name, p.Format)}
		}
	}
	return nil
}
```

Things this plan deliberately defers:

- **No** real ops code, no CLI, no MCP server. Plan B.
- **No** GitHub Actions workflows. Plan D.
- **No** `i18n-keys` or `oas-units` pre-commit hooks. Plan B wires them after their inputs exist.

- **No** real i18n catalogs (`en.toml`, `nb.toml`). Plan B authors them from the template seed.

Risks identified during planning that the implementer should know:

- **Go 1.26 availability.** If `pkgs.go_1_26` is missing in the pinned nixpkgs channel, the flake falls back to `go_1_25`. The plan permits this; surface the divergence to the user.
- **ogen non-determinism.** Modern ogen is deterministic given the same inputs, but if `codegen-clean` reports a diff on regeneration, do not paper over by committing the regenerated bytes — investigate the version pin.
- **`fiken-spec-lint` finds violations.** Plausible: Fiken's spec is good but not perfect. Surface to user before lowering the lint to a warning.
- **`fiken.NewClient` name drift.** If ogen renamed the constructor, the smoke test in Task 14 must be updated to the actual symbol. Do not delete the assertion — find the right name.
